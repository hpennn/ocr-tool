# 图片文字提取 - 免费在线 OCR 工具

纯前端实现的在线 OCR 文字识别工具，基于 Tesseract.js，无需服务器，隐私安全。

## ✨ 功能特点

- 📤 **多种上传方式**：拖拽上传、点击选择、Ctrl+V 粘贴
- 🌐 **多语言支持**：简体中文、繁体中文、英文
- 📋 **一键复制**：识别结果可编辑后复制
- 🔒 **隐私安全**：纯前端处理，图片不会上传到任何服务器
- 📱 **响应式设计**：手机、平板、电脑均可使用
- 💰 **完全免费**：无使用限制，无需注册登录

## 🛠️ 技术栈

- **HTML5 + Tailwind CSS**：现代化 UI
- **Tesseract.js**：纯前端 OCR 引擎（CDN 引入）
- **Cloudflare Pages**：全球 CDN 部署

## 📦 本地开发

```bash
# 直接打开 index.html 即可使用
# 或使用任意静态文件服务器
npx serve .
```

## 🚀 部署

本项目为纯静态站点，可直接部署到任何静态托管服务：

- Cloudflare Pages
- Vercel
- Netlify
- GitHub Pages
